# Gamification System

## XP

* Lesson complete: +10
* Correct answer: +2
* Perfect: +5 bonus

---

## Level

* Increasing XP requirement
* Example:

  * L1→L2: 50 XP
  * L2→L3: 75 XP

---

## Streak

* Daily learning required
* Reset if missed
* Bonus XP for streak

---

## Optional

* Lives system
* Rewarded ads (double XP, refill)
