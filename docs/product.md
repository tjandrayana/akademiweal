# Product Overview

## Vision

A gamified mobile app to learn investment basics in a fun and simple way.

---

## Target User

* Beginner (no prior knowledge)
* Age 18–35
* Afraid to start investing

---

## Core Features (MVP)

* Learning path (level-based)
* Lesson + quiz
* XP, level, streak
* Reward system
* Ads + premium (soft monetization)

---

## Non-Goals

* No trading
* No advanced analytics
* No social features

---

## Success Metrics

* D1 retention > 40%
* D7 retention > 20%
* Avg session ≥ 3 min
